How to use:

1. Compile an exe file: dart compile exe copyfile.exe
2. Run program: copyfile.exe <first file name> <second file name>

Case #1: copyfile.exe file1.txt file2.txt
Case #2: copyfile.exe file.txt file2.txt
Case #3: copyfile.exe firstFile.txt secondFile.txt
Case #4: (In this case you created an second file) copyfile.exe first.txt second.txt
